拍场模板组件

快速搭建拍场类型模板页面。

如何使用#

首先执行 yarn 或 npm install，确保 plop 已被安装。然后通过 plop 构建模板组件。

yarn plop screen
# 或者
npm run plop screen

命令行交互#

问题设置#

  1. 输入页面名称：输入新建的拍场页面名称，采用小驼峰命名规范。
  2. 选择项目模块：选择新建的拍场页面，放在哪个模块。目前有alita、batman、conan、doraemon。
  3. 选择页面模板：选择拍场页面模板文件。目前有粘性吸顶模板、固定吸顶模板。


生成模板页面#

脚手架会在指定模块下创建文件目录，在路由文件中配置对应路由。

预留如下UI区域#

  • 路由导航区
  • 顶部banner区
  • 粘性定位区
  • FlatList列表区
    • FlatList renderItem

使用者可以根据自己的业务需求，填充、替换、删除以上区域。

集成如下常见功能#

  • 上拉加载
  • 下拉刷新
  • 粘性动画定位

使用者传入对应参数即可使用。

Props#

模板页面是基于StickyScreen组件进行开发，其属性如下：

属性说明类型
scrollViewConfig自定义外层scrollView的属性，属性项同RN原生一致。ScrollViewProps
flatListConfig自定义内层FlatList的属性，属性项同RN原生一致。FlatListProps
listContainerStyle设置列表或children区域容器的样式。ViewStyle
headerView自定义页面顶部header UI视图。JSX Element
stickyView自定义页面粘性定位 UI视图。JSX Element
children组件标签可接收自定义子组件，传入此属性则会覆盖内部默认的FlatList。JSX Element
onEndReached上拉加载回调函数，列表滚动到底部时调用。function
onRefresh下拉刷新回调函数，页面滚动到顶部下拉时调用。需与isRefreshing属性配合使用。function
isRefreshing设置为true时，列表顶部显示一个正在加载的符号。boolean
