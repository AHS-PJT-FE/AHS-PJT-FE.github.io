Tooltips

tooltips方便快速接入提示框

使用方法#

<Tooltip
popWidth={272}
popHeight={64}
pointerColor={"rgba(0, 0, 0, 1)"}
backgroundColor={"rgba(0, 0, 0, 1)"}
pointDirection={"bottom"}
popoverDirection={"left"}
overlayColor={"transparent"}
pointDxy={38}
arrowDirection={"center"}
withPointer={true}
withOverlay={true}
popover={ // 弹窗样式
<Text
style={{
color: "white",
marginVertical: 12,
marginHorizontal: 16,
fontSize: 14,
lineHeight: 20,
textAlign: "left"
}}
>
对质检方为“爱回收或商家”的物品出价您可选择具体的质检方
</Text>
}
>
{//具体弹窗依附的view}
</Tooltip>

Props#

withPointer: boolean;#

是否显示箭头

popover: React.ReactElement;#

弹窗UI

popHeight: number;#

弹窗高度

popWidth: number;#

弹窗宽度

pointerColor: string;#

箭头颜色

onClose: () => void;#

关闭回调

onOpen: () => void;#

打开回调

withOverlay: boolean;#

是否展示弹窗背景

overlayColor: string;#

弹窗背景颜色

pointDirection: "top" | "bottom";#

箭头位置可以使在弹窗上面或下面

popoverDirection: "left" | "right";#

弹窗居左还是居右显示

pointDxy?: number;#

箭头距左边或右边的偏移

arrowDirection: "center";#

箭头展示位置，居中展示

autoShowTips: boolean;#

是否自动展示提示，否则点击显示

showTooltips: boolean;#

是否展示提示

pointerPadding?: number;#

箭头距内容距离

onItemClick?: () => void;#

内容点击事件回调

contentMargin?: number;#

弹窗提示左右边距
