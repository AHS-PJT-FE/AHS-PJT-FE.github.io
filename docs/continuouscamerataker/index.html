入参模型#

public class PicDescription {
private long id; // 物品ID
private int position; // 拍照项位置
private String sampleImage; // 拍摄示例图片,为空时不展示右上角的预览图片
private String localPicturePath; // 本地拍摄完成后返回图片
private String promptTitle; // 顶部title文案
private String promptDescription; // 顶部描述文案
public int getPosition() {
return position;
}
public void setPosition(int position) {
this.position = position;
}
public long getId() {
return id;
}
public void setId(long id) {
this.id = id;
}
public String getSampleImage() {
return sampleImage;
}
public void setSampleImage(String sampleImage) {
this.sampleImage = sampleImage;
}
public String getLocalPicturePath() {
return localPicturePath;
}
public void setLocalPicturePath(String localPicturePath) {
this.localPicturePath = localPicturePath;
}
public String getPromptTitle() {
return promptTitle;
}
public void setPromptTitle(String promptTitle) {
this.promptTitle = promptTitle;
}
public String getPromptDescription() {
return promptDescription;
}
public void setPromptDescription(String promptDescription) {
this.promptDescription = promptDescription;
}
}

连续拍照模式#

参数类型

  • setOptTakeType

    枚举值, OptTakeType

public enum OptTakeType {
OPT(0), // 连续拍照模式
PAIPAI(1); // 单张拍摄模式
private int type;
OptTakeType(int type) {
}
}

  • useLargestSize ()

    Booblean 是否拍摄大图
    true：选择当前相机最多分辨率拍摄
    false：快照模式拍摄

  • setMaxWidth

    (设置最大拍摄尺寸)

  • setOnPhotoTakenCallback

    设置连续拍照Callback回调

  • takePic

    传入拍照参数并跳转至拍摄页面
    activity : Activity 当前activity
    descriptionList : ArrayList-PicDescription 传入的拍照信息

new OptPic().setOptTakeType(OptTakeType.OPT)
.useLargestSize(true)
.setMaxWidth(1920)
.setOnPhotoTakenCallback(new IOnPhotoTakenCallback() {
@Override
public void onPhotoTaken(PicDescription picDescription) {
descriptionList.add(picDescription);
if (descriptionList.size() == descriptions.size()) {
// 先跳转页面，再关闭拍照页
Intent intent = new Intent(MainActivity.this, TestActivity.class);
intent.putExtra("PATH", picDescription.getLocalPicturePath());
startActivity(intent);
}
}
})
.takePic(MainActivity.this, descriptions);

单张拍照模式#

参数类型

  • setOptTakeType

    枚举值, OptTakeType

public enum OptTakeType {
OPT(0), // 连续拍照模式
PAIPAI(1); // 单张拍摄模式
private int type;
OptTakeType(int type) {
}
}

  • useLargestSize ()

    Booblean 是否拍摄大图
    true：选择当前相机最多分辨率拍摄
    false：快照模式拍摄

  • setMaxWidth

    (设置最大拍摄尺寸)

  • setOnPaiPaiPhotoTakenCallback

    设置单张拍照Callback回调

public interface IOnPaiPaiPhotoTakenCallback {
void onPhotoTaken(WeakReference<Activity> activityWeakReference, PicDescription picDescription);
}

  • takePic

    传入拍照参数并跳转至拍摄页面
    activity : Activity // 当前activity
    descriptionList : ArrayList-PicDescription // 传入的拍照信息

new OptPic().setOptTakeType(OptTakeType.PAIPAI)
.useLargestSize(true)
.setMaxWidth(1920)
.setOnPaiPaiPhotoTakenCallback(new IOnPaiPaiPhotoTakenCallback() {
@Override
public void onPhotoTaken(WeakReference<Activity> activityWeakReference, PicDescription picDescription) {
descriptionList.add(picDescription);
// 先跳转页面，再关闭拍照页
Intent intent = new Intent(MainActivity.this, TestActivity.class);
intent.putExtra("PATH", picDescription.getLocalPicturePath());
startActivity(intent);
activityWeakReference.get().finish();
}
})
.takePic(MainActivity.this, descriptions);
