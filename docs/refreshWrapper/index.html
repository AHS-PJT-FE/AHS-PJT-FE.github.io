RefreshWrapper

背景#

1、目前项目中使用focus、blur的方式繁琐不统一，有的页面还在使用老的willFocus、willBlur的API导致不可控的bug出现

2、在混合栈的模式下，当RN跳转至Native的页面后从native页面回退到RN页面时会出现无法获取回退事件的情况，造成无法刷新页面，此时需要从native页面发送通知到RN页面比较繁琐

使用方法#

Component使用方法如下：

1、Component继承CommonPage，CommonPage中进行了子view的绑定以及如下3中生命周期调用的方法：

constructor(props)
{
super(props);
/**
* 自动绑定子view依赖简化刷新接入流程
*/
if (props.onRef) {
props.onRef(this);
}
this.bus = new EventBus();
this.setOptions();
}
/**
* 子view继承此函数可获得native跳回页面是刷新
*/
nativeRefreshPage(){}
/**
* 子view继承此函数可获得RN栈跳回focus刷新
*/
focusRefreshPage(e:any){}
/**
* 子view继承此函数可获得RN失去焦点blur刷新
*/
blurRefreshPage(e:any){}

2、根据需求选择自己需要复写的方法，以京东转卖为例具体如下：

/**
* 在页面获取焦点时调用
*/
focusRefreshPage(e:any) {
super.focusRefreshPage(e);
const {getLimitInfo} = this.publicAuctionStore;
this.props.limitInfo.isNewMerchant && getLimitInfo();
if (!this.publicAuctionStore.scrollInitY) {
this.publicAuctionStore.initScroll();
}
!this.socket && this.initSocket();
this.updateItems();
Platform.OS == "ios" &&
OptNativeModule.ifReactNativeStatusBarLight &&
OptNativeModule.ifReactNativeStatusBarLight(true);
}
/**
* 在页面失去焦点时调用
*/
blurRefreshPage(e:any){
super.blurRefreshPage(e);
Platform.OS == "ios" &&
OptNativeModule.ifReactNativeStatusBarLight &&
OptNativeModule.ifReactNativeStatusBarLight(false);
this.publicAuctionStore.hideToolTip();
}
/**
* native页面回退到当前页面时调用
*/
nativeRefreshPage(){
super.nativeRefreshPage();
}
/**
* 对原有的Component进行导出操作
*/
export default RefreshWrapper(PublicAuction);

React-Hooks使用方法如下：

1、在hook组件的最上层调用RefreshWrapperHook方法，RefreshWrapperHook参数如下：

interface RefreshWrapperHookProps {
/**
* 子view继承此函数可获得native跳回页面是刷新
* 传入类型为：()=>{}
*/
nativeRefreshPage?: any;
/**
* 子view继承此函数可获得RN栈跳回focus刷新
* 传入类型为：(event)=>{}
*/
focusRefreshPage?: any;
/**
* 子view继承此函数可获得RN失去焦点blur刷新
* 传入类型为：(event)=>{}
*/
blurRefreshPage?: any;
/**
* 页面路由名称
* 获取方式：props.route.name
*/
routeName?: String;
}

2、在hook组件的最上层调用RefreshWrapperHook方法方式如下：

export default function TestWrapperHook(props: any) {
/**
* 以自定义hook的方式导入
*/
RefreshWrapperHook({
nativeRefreshPage: () => {
console.log("\n\n\n\n============= nativeRefreshPage =============\n\n\n\n");
},
focusRefreshPage: e => {
console.log("\n\n\n\n============= focusRefreshPage =============\n\n\n\n");
},
blurRefreshPage: e => {
console.log("\n\n\n\n============= blurRefreshPage =============\n\n\n\n");
},
routeName: props.route.name
});
return (
<SafeAreaView>
<TouchableOpacity
style={{ marginTop: 100 }}
onPress={() => {
navigator.pagePush(props.navigation, "DarkAuctionList");
}}
>
<Text>test page </Text>
</TouchableOpacity>
</SafeAreaView>
);
}
